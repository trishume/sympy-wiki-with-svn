

## Command-Line Access

Use this command to check out the latest project source code:

```
git clone git://github.com/sympy/sympy.git
```

You can browse the repository online here: http://github.com/sympy/sympy/
as well as find there other options to download it (http transport, tgz, zip, ...)


You may also go to the original googlecode [http://code.google.com/p/sympy/source/checkout Source tab].
